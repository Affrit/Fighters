# stage-2-es6-for-everyone

## Instalation

`npm install`

`npm run build`

`npm run start`

open http://localhost:8080/

## Controls

`for first fighter:`
`key: A - hit`
`key: D - block`
`keys: Q,W,E - critical hit`

`for second fighter:`
`key: J - hit`
`key: L - block`
`keys: U,I,O - critical hit`
